# 오더캐치 (ORDER_CATCH) — 브랜드 소개 웹

문자로 들어오는 주문을 AI가 자동으로 발주 리스트로 정리하는 **AI SMS 오더 플랫폼** 오더캐치의 IR 시연용 브랜드 소개 사이트.

## 구성
- `index.html` — 홈
- `features.html` — 기능 + 작동 방식
- `pricing.html` — 요금제
- `css/` — tokens.css(디자인 토큰), style.css
- `js/app.js` — GNB·스크롤 등장·인터랙션
- `assets/` — 로고/그래픽

정적 사이트. 로컬 실행: `python3 -m http.server 4173` 후 http://localhost:4173
